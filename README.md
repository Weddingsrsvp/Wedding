# Wedding RSVP Platform

A professional bilingual (Arabic & English) wedding invitation and RSVP platform.

## Features
- Elegant wedding invitation website
- Arabic and English support
- RSVP form with attendance tracking
- Guest count selection
- Countdown timer
- Photo gallery
- Google Maps integration
- QR code support
- Mobile-friendly responsive design
- Google Sheets integration for RSVP management

## Tech Stack
- HTML5
- CSS3
- JavaScript
- GitHub Pages
- Google Sheets API
